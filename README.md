# gnome-shell-extension-netbird

## About
This is a gnome extension to start, stop and configure netbird.

### Features
* Start and stop Netbird

## Requirements
* GNOME >= 46


## Installation
* Install extension
```bash
gnome-extensions install --force netbird@adfinis.com.shell-extension.zip
```
* Logout from GNOME desktop environment
